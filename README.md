## Description
This is a REST api using [Express](https://expressjs.com/)

## Installation

```bash
$ npm install
```

# Seeder
```bash
# import 
$ npm run data:import
# destroy
$ npm run data:destroy 

## Running the app

```bash
# development
$ node server

# watch mode
$ npm run server


```
